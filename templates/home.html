About

Holy catchphrase, Batman! I recently came across these two different sites, and figured I'd rescue the original Adam West Batman data from 1990's tables and unordered lists. The entire project is open-source and up on my Github account if you want to see the code used to make it (it's a small Flask site running on Heroku).

API

If you're looking to hack on the data with a JSON API, then you're in luck — I've exposed both a random endpoint and regular endpoint. The regular endpoint can be used to only grab Robin catchphrases or action words, as well.

The JSON data is also available on the project's Github page, too.
